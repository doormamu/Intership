# HTTP Task Server 
![Go Version](https://img.shields.io/badge/go-1.23-blue)

## О проекте
REST API сервис на Go, имитирующий систему асинхронной обработки задач.
Пользователь отправляет данные на сервер, получает уникальный ID (UUID) и может проверять статус выполнения задачи. Проект реализует многослойную архитектуру (Transport layer / Storage layer).

## Стек технологий
* **Язык:** Go 1.23
* **Маршрутизация:** Chi v5 (быстрый и легковесный роутер)
* **Хранилище:** In-memory (RAM) с использованием мап
* **Формат данных:** JSON

## API Эндпоинты

| Метод | Путь | Описание |
| :--- | :--- | :--- |
| `POST` | `/task` | Создать новую задачу. Принимает JSON `{"value": "..."}`. Возвращает UUID. |
| `GET` | `/status` | Проверить статус. Параметр `?key=UUID`. Возвращает `in progress` или `ready`. |
| `GET` | `/result` | Получить результат обработки. Параметр `?key=UUID`. |
