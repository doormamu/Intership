**–†–µ–∞–ª–∏–∑–æ–≤–∞–Ω–Ω—ã–µ –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã:**
* **–°–ª–æ–∏:** `Conv2D` (—Å–≤–µ—Ä—Ç–∫–∞), `MaxPooling`, `Dense` (–ø–æ–ª–Ω–æ—Å–≤—è–∑–Ω—ã–π), `Flatten`, `Dropout`.
* **–ê–∫—Ç–∏–≤–∞—Ü–∏–∏:** `ReLU`, `Softmax`.
* **–ù–æ—Ä–º–∞–ª–∏–∑–∞—Ü–∏—è:** `BatchNorm` (Batch Normalization).
* **–û–ø—Ç–∏–º–∏–∑–∞—Ç–æ—Ä—ã:** `SGD`, `SGD Momentum`.

### üíª –ü—Ä–∏–º–µ—Ä –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä—ã

–ö–æ–¥ –∏–Ω–∏—Ü–∏–∞–ª–∏–∑–∞—Ü–∏–∏ —Å–≤–µ—Ä—Ç–æ—á–Ω–æ–π —Å–µ—Ç–∏, –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–Ω–æ–π –¥–ª—è —Ç–µ—Å—Ç–æ–≤:
