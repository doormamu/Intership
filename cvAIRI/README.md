# Computer Vision & Deep Learning Lab

### Project List

| ID | Project Name | Tech Stack | Description |
| :--- | :--- | :--- | :--- |
| **01** | **[Image Alignment](./1hw)** | ![NumPy](https://img.shields.io/badge/NumPy-013243?logo=numpy&logoColor=white) | **Prokudin-Gorskii Colorization.** Restoring historical color images by aligning R, G, B channels using cross-correlation and pyramid search. |
| **02** | **[Bayer Demosaicing](./2hw)** | ![Python](https://img.shields.io/badge/Python-3776AB?logo=python&logoColor=white) | **RAW to RGB.** Implementation of bilinear interpolation and advanced demosaicing algorithms to reconstruct color images from CFA patterns. |
| **05** | **[DL Framework](./5hw)** | ![NumPy](https://img.shields.io/badge/NumPy-Core-orange) | **CNN from Scratch.** Implementation of a Deep Learning framework (Forward/Backward pass, Conv2D, MaxPooling, SGD) using only pure NumPy. |
| **06** | **[Face Regression](./6hw)** | ![PyTorch](https://img.shields.io/badge/PyTorch-EE4C2C?logo=pytorch&logoColor=white) | **Facial Keypoints Detection.** Training a Convolutional Neural Network (CNN) to predict 14 facial landmarks (eyes, nose, mouth) on images. |
| **11** | **[Object Tracking](./11hw)** | ![PyTorch](https://img.shields.io/badge/PyTorch-EE4C2C?logo=pytorch&logoColor=white) | **Multi-Object Tracking.** A hybrid tracking system combining **SSD** (Single Shot Detector) and **Correlation Filters** for efficient video processing. |
| **12** | **[GANs](./12hw)** | ![Lightning](https://img.shields.io/badge/Lightning-792EE5?logo=pytorch-lightning&logoColor=white) | **Sneaker Generation.** Using DCGAN and Super-Resolution GANs to generate photorealistic images of sneakers from noise. |

* **Deep Learning:** PyTorch, PyTorch Lightning
